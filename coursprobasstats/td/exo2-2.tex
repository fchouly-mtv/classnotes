\begin{exercice}\label{exo2-2} (\coolexo)

Montrer, via changement de variable, que les mod\`eles suivants sont bien lin\'eaires:

\begin{enumerate}

\item $Y_i = \beta_1 \ln x_i + \beta_0 + \varepsilon_i$.

\item $Y_i = \beta_2 x_i^2 + \beta_1 x_i + \beta_0 + \varepsilon_i$.

\item $Y_i = \beta_0 x_i^{\beta_1}\varepsilon_i$ (mod\`ele de Cobb-Douglas).

\item $Y_i = e^{\beta_1 x_i + \beta_0 + \varepsilon_i}  (1+e^{\beta_1 x_i + \beta_0 + \varepsilon_i})^{-1}$ (mod\`ele logistique).

\end{enumerate}

\end{exercice}
