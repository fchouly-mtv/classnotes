\begin{corrige}{3-13}

On applique 3 fois de suite la règle vue à l'exercice précédent:
\begin{eqnarray*}
 & & P ( A \cup B \cup C)\\
 & = & P ( A \cup B ) + P(C) - P ( (A\cup B) \cap C )\\
 & = & ( P(A) + P(B) + P ( A \cap B ) ) + P(C)\\
 & & - P ( (A\cap C) \cup (B \cap C ) )\\
 & = & ( P(A) + P(B) + P ( A \cap B ) ) + P(C)\\
 & & - ( P (A\cap C) + P(B \cap C) -  P ( (A\cap C) \cap (B \cap C ) ) )
\end{eqnarray*}
Il suffit finalement de réordonner l'expression obtenue, et de remarquer que $$P ( (A\cap C) \cap (B \cap C ) ) = P ( A \cap B \cap C ).$$

\end{corrige}
