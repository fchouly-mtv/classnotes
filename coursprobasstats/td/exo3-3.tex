\begin{exercice}\label{exo3-3} (\minsyndical)

Soit $(\Omega, P)$ un espace probabilis\'e, avec $P$ une mesure
de probabilit\'e sur les \'ev\`enements de $\Omega$. Montrer les propositions suivantes :

\begin{enumerate}

\item Pour tout \'ev\`enement $A$ :
\[
P(\bar{A}) = 1 - P(A).
\]

\item Pour tous les \'ev\`enements $A$ et $B$ :
\[
P(A \cup B) = P(A) + P(B) - P(A \cap B).
\]

{\it Indication : faire un dessin !}

\item En d\'eduire l'in\'egalit\'e de Bonferroni :
\[
P(A \cup B) \geq P(A) + P(B) - 1.
\]

\end{enumerate}

\end{exercice}
